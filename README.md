# Web stopwatch

A modern, responsive web stopwatch with keyboard shortcuts, lap tracking, and multiple visual themes. Designed to be simple, clean, and professional.

## Features
- Start, Stop, Reset, Lap via buttons or keyboard:
  - Enter → Start
  - Space → Stop
  - Backspace → Reset
  - L → Lap
- Multiple visual themes
- Lap history display
- Clean, responsive UI with gradients
